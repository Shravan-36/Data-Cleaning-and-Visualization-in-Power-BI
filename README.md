📊 Data Cleaning and Visualization in Power BI

📌 Project Overview
This project demonstrates how raw data can be cleaned, transformed, and visualized using Power BI. The dataset was imported into Power BI where Power Query was used to clean and prepare the data before creating interactive dashboards and visualizations.
The project highlights how data can be transformed from a raw dataset into meaningful insights using Power BI’s data preparation and visualization capabilities.

🎯 Project Objective
1.Clean and transform raw data using Power Query
2.Prepare the dataset for analysis
3.Create interactive dashboards and visualizations
4.Identify patterns and trends within the dataset

📂 Dataset
The dataset used in this project was sourced from:

🛠️ Tools & Technologies Used
* Power BI
* Power Query
* Data Modeling
* Interactive Dashboard Visualizations

⚙️ Project Workflow
-Imported the dataset into Power BI.
-Cleaned and transformed the dataset using Power Query Editor.
-Removed duplicates, handled missing values, and corrected data types.
-Created multiple visualizations and charts to analyze the dataset.
-Designed an interactive dashboard to present insights.

📊 Dashboard Features
* Interactive charts and graphs
* KPI indicators
* Trend analysis
* Data filtering and segmentation

💡 Skills Demonstrated
-Data cleaning and transformation
-Data visualization and dashboard creation
-Data analysis using Power BI
-Turning raw data into meaningful insights

📖 Learning Outcome
